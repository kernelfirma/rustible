# Security Policy

## Supported Versions

Rustible is in alpha. Security fixes are provided for the latest commit on the
main branch and the most recent tagged release.

## Reporting a Vulnerability

Please report vulnerabilities privately if possible.

- Use GitHub Security Advisories (Security tab) to open a private report.
- If private reporting is not available, open a GitHub issue with minimal
  details and request a private follow-up channel.
- If you need a direct contact, reach out to @adolago via GitHub.

Include the affected version, reproduction steps, and impact assessment when
available.

## Response Process

We aim to acknowledge reports within 3 business days and provide status updates
as we triage and develop a fix.
